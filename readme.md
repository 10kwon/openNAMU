## 개요
Ownet을 파이썬으로 재 구현한 버전 입니다.

## 설치법
set 폴더에 있는 set.json을 밖으로 꺼낸 다음 안에 있는 내용을 적절하게 수정하고 app.py를 파이썬 3.x 버전으로 실행하면 됩니다. (파이썬 3.6을 권장 합니다.)
